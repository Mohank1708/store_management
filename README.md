# Store Management System

A role-based inventory management system for restaurants.

## Features
- Manager: View inventory, add/edit/delete items
- Purchase Manager: Add purchased items, upload Excel files
- Kitchen Manager: Send items to kitchen

## Local Setup
```bash
pip install -r requirements.txt
python app.py
```
